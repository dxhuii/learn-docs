# 字符串函数
通过 sql 指令对字符串进行处理

## concat() 拼接字符串
灵活拼接字符串
``` sql
select concat(stu_name,' - ',stu_gender, ' = ', stu_age) from students;
+---------------------------------------------------+
| concat(stu_name,' - ',stu_gender, ' = ', stu_age) |
+---------------------------------------------------+
| 张三 - 男 = 21                                    |
| 李四 - 男 = 22                                    |
| 王五 - 男 = 23                                    |
+---------------------------------------------------+
```

## concat_ws() 拼接字符串
多字段拼接，字段间使用同样的拼接符(会自动跳过NULL)
``` sql
select concat_ws(' ~ ', stu_name, stu_gender, stu_age) from students;
+-------------------------------------------------+
| concat_ws(' ~ ', stu_name, stu_gender, stu_age) |
+-------------------------------------------------+
| 张三 ~ 男 ~ 21                                  |
| 李四 ~ 男 ~ 22                                  |
| 王五 ~ 男 ~ 23                                  |
+-------------------------------------------------+
```

## upper() 小写转大写
``` sql
select upper(stu_name) from stus;
+-----------------+
| upper(stu_name) |
+-----------------+
| OBJ             |
| OMG             |
| HELLO           |
+-----------------+
```

## lower() 大写转小写
``` sql
select lower(stu_name) from stus;
+-----------------+
| lower(stu_name) |
+-----------------+
| obj             |
| omg             |
| hello           |
+-----------------+
```

## substring() 截取字符串
`注意`：数据库中索引从 `1` 开始
``` sql
select stu_name,substring(stu_tel,8,4) from stus;
+----------+------------------------+
| stu_name | substring(stu_tel,8,4) |
+----------+------------------------+
| 张三     | 3030                   |
| 李四     | 3031                   |
| 王五     | 3032                   |
+----------+------------------------+
```
