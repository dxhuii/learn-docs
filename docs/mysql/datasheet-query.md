# DQL 数据表字段查询
``` sql
-- select 关键字后指定要显示查询到的记录的哪些列
select <columnName1,columnName2...> from <tableName>;

-- 查询所有列（在项目中不建议使用 *）
select * from <tableName>;
```

## where 语句
- 在删除、修改及查询的语句后面都可以加 where 语句（条件）用于筛选满足特定的添加的数据进行删除、修改、和查询操作
``` sql
-- = 等于
select * from stus where stu_num = '1';

-- != 不等于/<>大于或者小于
select * from stus where stu_num != '1';
select * from stus where stu_num <> '1';

-- > 大于
select * from stus where stu_age > 1;

-- < 小于
select * from stus where stu_age < 1;

-- >= 大于等于
select * from stus where stu_age >= 1;

-- >= 小于等于
select * from stus where stu_age <= 1;

-- between and 区间查询，查询年龄在 18-20 之间的
select * from stus where stu_age between 18 and 20;
```

## between 多条件查询
``` sql
-- and 并且，筛选多个条件同时满足的记录
select * from stus where stu_gender = '女' and stu_age > 1;

-- or 或者，筛选多个条件中至少一条满足的记录
select * from stus where stu_gender = '女' or stu_age > 1;

-- not 取反，查询年龄不在 18-20 之间的
select * from stus where stu_age not between 18 and 20;
```

## like （模糊查询）
- 在 like 关键字后的 reg 表达式中 `%` 表示任意多个字符 [%o% 包含字母o]
- `%` 表示任意一个字符 [_o% 第二个字母为 o]
``` sql
-- 查询名字中包含 o 的数据
select * from stus where stu_name like '%o%';

-- 查询名字第一个字为张的数据
select * from stus where stu_name like '张%';

-- 查询名字最后一个字为o的数据
select * from stus where stu_name like '%o';

-- 查询名字第二个字为o的数据
select * from stus where stu_name like '_o';
```

## 查询结果处理
- 设置查询的列：声明显示查询结果的指定列
``` sql
select colimnName1,colimnName2... from stus where stu_age>20;
```

- 计算列：从数据表中查询的记录的列，进行一定的运算之后显示出来的列
``` sql
-- 出生年份 = 当前年份 - 年龄
select stu_name, 2023-stu_age from stus;
+----------+--------------+
| stu_name | 2023-stu_age |
+----------+--------------+
| 张三     |         2003 |
| 李四     |         2003 |
| 王五     |         2003 |
+----------+--------------+
```

- 字段别名 `as`：给查询结果的列名取一个语义性更强的别名，(如下案例中 `as` 关键字也可以省略)
``` sql
select stu_name, 2023-stu_age as stu_birth_year from stus;
+----------+----------------+
| stu_name | stu_birth_year |
+----------+----------------+
| 张三     |           2003 |
| 李四     |           2003 |
| 王五     |           2003 |
+----------+----------------+

select stu_name as 姓名, 2023-stu_age as 出生年份 from stus;
+--------+--------------+
| 姓名   | 出生年份     |
+--------+--------------+
| 张三   |         2003 |
| 李四   |         2003 |
| 王五   |         2003 |
+--------+--------------+
```

- 消除重复行 `distinct`：从查询结果出消除重复的数据
``` sql
select stu_age from stus;
+---------+
| stu_age |
+---------+
|      20 |
|      20 |
|      20 |
+---------+

select distinct stu_age from stus;
+---------+
| stu_age |
+---------+
|      20 |
+---------+
```

## 排序操作 `order by`
- 将查询到的满足条件的记录按照指定列的值升序/降序排列

    - desc：降序
