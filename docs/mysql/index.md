# [MySQL](https://www.mysql.com/downloads/)
MySQL 是目前最为流行的开放源码的数据库，是完全网络化的跨平台的关系型数据库系统，它是由瑞典 MySQLAB 公司开发，目前属于 Oracle 公司。任何人都能从 Internet 下载 MySQL 软件，而无需支付任费用，并且“开放源码”意味着任何人都可以使用和修改该软件
