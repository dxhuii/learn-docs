# 字符串案例

## 字符串转数组
正常字符串
```js
const str = '1, 2, 3, 4, 5'
console.log(str.split(','))
// ['1', '2', '3', '4', '5']
```

复杂字符串，key看到下面的 `张弦[董事, 经理]` 分隔出现了问题
```js
const str = '王喆[董事长], 张弦[董事, 经理], 魏弘[董事], 乐欢[监事]'
console.log([...str])
// ['王喆[董事长]', '张弦[董事', ' 经理]', '魏弘[董事]', '乐欢[监事]']
```
修改为
``` js
const str = '王喆[董事长], 张弦[董事, 经理], 魏弘[董事], 乐欢[监事]'
const arr = str.split(/(?<=]),/g)
// ['王喆[董事长]', ' 张弦[董事, 经理]', ' 魏弘[董事]', ' 乐欢[监事]']
```
