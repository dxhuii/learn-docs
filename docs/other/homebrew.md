# Homebrew
https://blog.csdn.net/Jimmmyking/article/details/126105788

| 命令        |      作用      |
| ------------- | :----------- |
| brew help      | 查看帮助 |
| brew -v      | 查看版本 |
| brew update      | 更新 brew |
| brew install xxx      | 安装包 |
| brew uninstall      | 卸载包 |
| brew outdated      | 查询可更新的包 |
| brew cleanup      | 清理旧版本 |
| brew list      | 查看安装列表 |
