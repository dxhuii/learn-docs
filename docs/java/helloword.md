# 第一个应用程序

## HelloWord
- 文件名称必须与代码中的类名称一致

```java
public class HelloWord {
    public static void main(String[] args){
        System.out.println("Hello Word");
    }
}
```

## 编译 javac 文件名
```
javac HelloWord.java
```

## 运行 java 类名
```
java HelloWord
```
