# SignaturePad 电子签名

## 安装 [官网配置项](https://github.com/szimek/signature_pad)
``` js
pnpm i signature_pad
```

## Props

| 参数      | 说明                                            | 类型   | 可选值 | 默认值    |
| :-------- | :---------------------------------------------- | :----- | :----- | :-------- |
| width | 宽度                                        | number | -      | 500 |
| height   | 高度                                        | number | -      | 300 |
| customStyle    | 自定义 canvas 样式                                        | object | -      | -         |
| backgroundColor       | 背景色                            | string | -      | 'rgb(240, 240, 240)'  |
| penColor       | 画线的颜色                                    | string | -      | '#000'        |
